## Улучшение сходимости нейросетей
*Цель*: Необходимо реализовать оставшиеся методы оптимизации, регуляризации и сравнить их

План работы:
- Реализация Xavier
- Реализация DropConnect
- Сравнение обучения при использовании Dropout и DropConnect
- Реализация LayerNorm

Используемые технологии:
- Pytorch
- Sklearn
- Matplotlib
- Numpy
